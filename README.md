Descripción del Proyecto
Juego de batallas por turnos desarrollado en Java Swing con arquitectura MVC, utilizando MySQL como base de datos para persistir personajes, batallas y estadísticas.

Tecnologías Usadas
- Java, Java Swing para la interfaz, JDBC en la conectividad con la base de datos,MySQL para la base de datos, NetBeans IDE para el desarollo y una arquitectura MVC
